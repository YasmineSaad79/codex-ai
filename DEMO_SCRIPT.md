# 🎬 سيناريو تصوير الديمو للجامعة

## 📋 التحضير قبل التصوير

1. شغل التطبيق:
```bash
cd streamlit_chatbot
streamlit run chat_app.py
```

2. افتح المتصفح على: `http://localhost:8501`
3. امسح أي محادثات قديمة (زر مسح المحادثة)

---

## 🎥 سيناريو التصوير (3-5 دقائق)

### المشهد 1: المقدمة (30 ثانية)
**ماذا تقول:**
> "السلام عليكم، اليوم راح أعرض لكم مشروع Chatbot ذكي لشركة الصيف، مبني باستخدام Streamlit و Python"

**ماذا تفعل:**
- اعرض الشاشة الرئيسية للتطبيق
- وضح العنوان والواجهة

---

### المشهد 2: شرح الواجهة (45 ثانية)
**ماذا تقول:**
> "التطبيق يحتوي على واجهة بسيطة وسهلة، في الجانب عندنا الإعدادات"

**ماذا تفعل:**
1. اضغط على الشريط الجانبي (Sidebar)
2. وضح خيار اللغة
3. وضح زر مسح المحادثة
4. وضح عداد الرسائل

---

### المشهد 3: تجربة المحادثة بالعربية (90 ثانية)
**ماذا تقول:**
> "الآن راح نجرب المحادثة بالعربية"

**ماذا تكتب بالترتيب:**

1. **الرسالة الأولى:**
   ```
   مرحبا
   ```
   **انتظر الرد** ثم قل: "البوت يرحب فينا ويسأل كيف يقدر يساعد"

2. **الرسالة الثانية:**
   ```
   من أنت؟
   ```
   **انتظر الرد** ثم قل: "يعرف نفسه كمساعد ذكي للشركة"

3. **الرسالة الثالثة:**
   ```
   ما هي منتجات الصيف؟
   ```
   **انتظر الرد** ثم قل: "يعطينا معلومات عن المنتجات"

4. **الرسالة الرابعة:**
   ```
   كيف أتواصل معكم؟
   ```
   **انتظر الرد** ثم قل: "يوفر طرق التواصل"

---

### المشهد 4: تغيير اللغة (60 ثانية)
**ماذا تقول:**
> "الآن راح نجرب نغير اللغة للإنجليزية"

**ماذا تفعل:**
1. اذهب للشريط الجانبي
2. غير اللغة من "العربية" إلى "English"
3. قل: "لاحظوا التطبيق يدعم اللغتين"

**ماذا تكتب بالإنجليزي:**

1. **الرسالة الأولى:**
   ```
   Hello
   ```
   **انتظر الرد**

2. **الرسالة الثانية:**
   ```
   What products do you sell?
   ```
   **انتظر الرد** ثم قل: "البوت يرد بالإنجليزي"

---

### المشهد 5: عرض المميزات (45 ثانية)
**ماذا تقول:**
> "من مميزات التطبيق:"

**ماذا تفعل:**
1. وضح الطوابع الزمنية (Timestamps) على الرسائل
2. اذهب للشريط الجانبي وأشر لعداد الرسائل
3. قل: "كل رسالة عليها وقت وتاريخ، والتطبيق يحسب عدد الرسائل"

---

### المشهد 6: مسح المحادثة (30 ثانية)
**ماذا تقول:**
> "نقدر نمسح المحادثة بسهولة"

**ماذا تفعل:**
1. اضغط على زر "مسح المحادثة"
2. وضح أن الشاشة رجعت فاضية
3. قل: "والآن نقدر نبدأ محادثة جديدة"

---

### المشهد 7: الخاتمة (30 ثانية)
**ماذا تقول:**
> "هذا كان عرض سريع للـ Chatbot. التطبيق مبني بـ Python و Streamlit، ويدعم اللغتين العربية والإنجليزية، وفيه نظام ردود ذكي يفهم الأسئلة المختلفة. شكراً لكم"

**ماذا تفعل:**
- اعرض الشاشة الكاملة مرة أخيرة
- يمكنك إظهار الكود بسرعة (اختياري)

---

## 💡 نصائح للتصوير

### قبل التصوير:
- ✅ نظف سطح المكتب
- ✅ أغلق التطبيقات غير الضرورية
- ✅ اخفي الإشعارات
- ✅ استخدم وضع ملء الشاشة للمتصفح (F11)
- ✅ جهز نص ما راح تقوله

### أثناء التصوير:
- 🎤 تكلم بوضوح وببطء
- ⏸️ اعمل وقفة قصيرة بعد كل رد من البوت
- 🖱️ حرك الماوس ببطء
- 📝 اكتب ببطء عشان يشوفون
- 😊 ابتسم وكن واثق

### بعد كل رسالة:
- انتظر 2-3 ثواني قبل الرسالة التالية
- اقرأ الرد بصوت عالي أو لخصه

- شرح كيف يتم التثبيت

## 📱 للتصوير من الموبايل

إذا تبي تصور من الموبايل:
1. شغل التطبيق على الكمبيوتر
2. اطلع الـ IP address:
   ```bash
   ipconfig
   ```
3. افتح من الموبايل: `http://[your-ip]:8501`
4. صور الشاشة
