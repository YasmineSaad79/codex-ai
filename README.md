# Al Saif Chatbot

تطبيق محادثة تفاعلي لشركة الصيف باستخدام Streamlit

## المميزات

- 💬 واجهة محادثة تفاعلية
- 🌐 دعم اللغتين العربية والإنجليزية
- 📝 حفظ سجل المحادثات
- ⏰ عرض الوقت والتاريخ لكل رسالة
- 🎨 تصميم عصري وسهل الاستخدام

## التثبيت

1. تثبيت المكتبات المطلوبة:
```bash
pip install -r requirements.txt
```

## التشغيل

```bash
streamlit run chat_app.py
```

سيتم فتح التطبيق في المتصفح على العنوان: `http://localhost:8501`

## الاستخدام

1. اختر اللغة من الشريط الجانبي
2. اكتب رسالتك في مربع الإدخال
3. اضغط Enter لإرسال الرسالة
4. يمكنك مسح المحادثة باستخدام زر "مسح المحادثة"

## التطوير المستقبلي

- إضافة قاعدة بيانات لحفظ المحادثات
- دمج نموذج AI متقدم (GPT, Claude, etc.)
- إضافة المزيد من الردود الذكية
- دعم الملفات والصور
- نظام تقييم الردود
